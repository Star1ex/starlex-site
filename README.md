# Starlex

A collaborative task management application designed to help teams organize, track, and complete their work efficiently.

## Features

- **Task Management**: Create, assign, and track tasks with due dates and priorities
- **Team Collaboration**: Work together with your team members in real-time
- **Project Organization**: Organize tasks by projects, teams, or custom categories
- **Progress Tracking**: Monitor task completion and team productivity
- **User-Friendly Interface**: Intuitive design for seamless task management

## Getting Started

### Prerequisites

- Go 
- Docker (docker-compose)
- Node.js (v18 or higher)
- npm or yarn package manager
- Python
## Usage

### Creating a Task

1. Click on "Add New Task" button
2. Fill in the task details (title, description, assignee, due date)
3. Set the priority level
4. Save the task

### Managing Tasks

- **Edit**: Click on a task to modify its details
- **Complete**: Mark tasks as done when finished
- **Delete**: Remove tasks that are no longer needed
- **Filter**: Use filters to view tasks by status, assignee, or date
